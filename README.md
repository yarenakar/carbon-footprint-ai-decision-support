# Carbon Footprint AI Decision Support System

This repository contains the source codes developed within the scope of the graduation project.

## Project Overview

This graduation project focuses on carbon footprint reduction in textile production. The project includes exploratory data analysis, data preprocessing, prediction modeling, causality analysis, and counterfactual analysis to support data-driven sustainability decisions.

The main aim of the project is to evaluate carbon emission behavior in textile production and support decision-making for carbon reduction interventions.

## Code Files

The project consists of four main Google Colab notebooks:

1. `01_kesifsel_veri_analizi.ipynb`  
   This notebook includes exploratory data analysis of the textile production carbon emission dataset.

2. `02_veri_on_isleme.ipynb`  
   This notebook includes data preprocessing steps such as missing value control, outlier analysis, feature preparation, and model-ready dataset creation.

3. `03_tahminleme_modelleri.ipynb`  
   This notebook includes machine learning models developed for carbon emission prediction.

4. `04_nedensellik_ve_karsiolgusal_analiz.ipynb`  
   This notebook includes causality and counterfactual analysis for evaluating the effects of sustainability interventions on carbon emissions.

## Data Note

The dataset used in this project is not shared publicly due to project and data privacy restrictions. The notebooks can be executed after updating the dataset path according to the user's local or Google Drive environment.
